# Gen-Z Portal - Bootcamp Project 🚀

A comprehensive Student Portal application developed using **Flutter**. This project was built to demonstrate proficiency in UI/UX design, state management, and backend REST API integration. 

## ✨ Features
* **User Authentication:** Secure Login and Registration flow.
* **Dashboard Overview:** Displays global stats and active course enrollments.
* **Course Management:** Browse categories, view all programs, and check detailed course descriptions.
* **Enrollment System:** Seamless 1-click enrollment into courses with real-time status updates.
* **Profile Management:** View and manage user profile details.
* **Local Storage:** Utilized **Hive** for fast, secure local storage of user sessions.

## 🛠️ Tech Stack
* **Frontend:** Flutter (Dart)
* **Design:** Material 3, Custom Dark Theme UI
* **Local Database:** Hive (NoSQL)
* **Backend Integration:** REST APIs (http package)

This project solidifies concepts in API handling, asynchronous programming in Dart, complex UI structuring, and navigation routing in a production-like environment.
